## Getting Started

### Prerequisites

- Java 21
- Gradle
- pre-commit
- Docker
- Kubernetes
- Skaffold

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/gudfhr95/open-ads.git
   cd open-ads
   ```
2. Install pre-commit hooks:
   ```sh
   pre-commit install
    ```
3. Build the project:
   ```sh
   ./gradlew build
   Run the application:
   ./gradlew run
   ```
