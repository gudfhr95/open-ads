## Development

### Prerequisites

- Java 21
- Gradle
- Docker
- Kubernetes
- Skaffold

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/gudfhr95/open-ads.git
   cd open-ads
   ```
2. Install pre-commit hooks
   ```sh
   pre-commit install
    ```

### Running Locally

```sh
skaffold dev
```
